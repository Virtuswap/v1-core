# README #

Virtuswap V1 smart contracts  
In-depth documentation is available at [docs.virtuswap.io](https://docs.virtuswap.io).

#  Local development #
Hardhat requires `node@>=16.0`


#  Installation #

```
git clone git@github.com:Virtuswap/v1-core.git
cd v1-core
npm i
```


# Compilation #
```
npx hardhat compile
```

# Running tests #
```
npx hardhat test
```
