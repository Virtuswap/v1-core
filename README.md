# README #

Virtuswap smart contracts

#  Installation #

```
git clone git@github.com:Virtuswap/v1-core.git
cd v1-core
sudo npm install -g truffle
npm i
```
Depending on your platform and setup, you may not need the `sudo`.

# Compilation #
```
truffle compile
```

# Running tests #
```
truffle test
```
